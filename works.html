02 : Project

I have worked on various Data Science projects, covering exploratory data analysis (EDA), machine learning model development, and interactive data visualization. In these projects, I have taken roles as a Data Engineer, Data Scientist, and Data Analyst, with the ability to work both independently and collaboratively in teams to build data-driven solutions.

I have implemented cloud-based solutions and automated ETL pipelines to efficiently process large-scale data, while also applying feature engineering and model evaluation techniques to enhance model performance. Through these projects, I strive to bridge the gap between raw data and actionable business insights.

Retail Store Analysis

Involved analyzing retail sales data across multiple product categories and regions to optimize business strategies and improve sales performance.

Sentiment Analysis

Building a Deep Learning model for analyzing customer sentiment from McDonald's reviews using Scikit-learn and TensorFlow.

Developing a Machine Learning model to predict used car prices based on various attributes, including technical specifications, car type, mileage, fuel consumption, etc.

Loan Approval Analysis

This project involved analyzing loan approval data to identify key factors influencing credit decisions.

Halal Check

A Deep Learning-based project that verifies the halal status of food products using OCR and NLP.

Fruits Classification

This project used MobileNetV2 to classify fruit images into five categories with high accuracy.

Recommendation System

This project develops a product recommendation system using Content-Based Filtering to provide personalized recommendations based on user preferences.
